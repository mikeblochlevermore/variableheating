Variable Heating

a CS50x project
🔥

We spent all our money on a Kronoterm heatpump...
so how can we afford to run it?!
This web app suggests a heating schedule that reduces the temperature (just a little) when the electricity prices are high.


⚡️

The app gets today's electricty prices from elpriserligenu.dk
Plus, if it's after 13.00, the prices for tomorrow.
Note: The zone is currently fixed to the east half of Denmark (undskyld Jylland...)


💶

Behind the scenes, the transport costs are added (defaulted to Cerius network),
factoring in winter or summer tarrifs, and the total costs are displayed.
Finally, the heating settings below are applied, to create a suggested schedule for the heatpump


🐍

Built with Python and Flask // View on GitHub
